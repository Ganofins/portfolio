RFID Based Billing System

RFID Based Billing System
We created a RFID based bill generation system to reduce the billing time in the malls. We built it using a Arduino, RFID Scanner, couple of RFID Tags and with couple cups of coffee. We created a GUI using Java and connected it with Arduino using a library, which fetched the data from the RFID tags using RFID Scanner to our GUI. It was effective and can reduce upto 34% time during bill generation.
